## 🎯 New Animation Style

### Entrance (When appearing):
- **Fade in** from 0% to 100% opacity
- **Scale in** from 90% to 100% size
- **Duration**: 300ms
- **Easing**: FastOutSlowInEasing (Material Design)

### Exit (When disappearing):
- **Fade out** from 100% to 0% opacity
- **Scale out** from 100% to 90% size
- **Duration**: 200ms (faster)
- **Easing**: LinearOutSlowInEasing

### Icon Selection:
- **Scale**: Selected = 100%, Unselected = 95%
- **Color**: Smooth 200ms transition
- **Duration**: 200ms
- **Easing**: FastOutSlowInEasing

### Label (Text):
- **Fade in**: 300ms when selected
- **Fade out**: 200ms when unselected
- **No expand/shrink** - just fade

## 🎬 Animation Timeline

### Selecting an item:
```
0ms   - Click
0-200ms - Icon scales 95% → 100%
0-200ms - Color changes to primary
0-300ms - Label fades in
200ms - Animation complete
```

### Deselecting an item:
```
0ms   - Click other item
0-200ms - Icon scales 100% → 95%

200ms - Animation complete
```
